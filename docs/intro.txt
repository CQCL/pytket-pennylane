pytket-pennylane
================

``pytket-pennylane`` is an extension to ``pytket`` that allows ``pytket``
backends and compilation passes to be used as
`PennyLane <https://pennylane.ai/>`_ devices.

``pytket-pennylane`` is available for Python 3.9, 3.10 and 3.11, on Linux, MacOS
and Windows. To install, run:

::

   pip install pytket-pennylane
